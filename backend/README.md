# Backend - Club Nova Frota

Área reservada para funções e automações do Club Nova Frota.

## Objetivo

Centralizar regras de negócio que não devem ficar diretamente no aplicativo.

## Funções futuras

- Criar ordem
- Aprovar comprovante
- Gerar pontos
- Atualizar ranking
- Enviar notificações
- Integrar parceiros e benefícios

## Tecnologia prevista

- Firebase Cloud Functions
- Firestore Triggers
- Firebase Cloud Messaging

## Status

Planejamento e arquitetura em andamento.
