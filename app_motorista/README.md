# App Motorista - Club Nova Frota

Aplicativo mobile destinado aos motoristas parceiros da Nova Frota.

## Objetivo

Permitir que o motorista acesse fretes, solicite ordens, acompanhe viagens, envie comprovantes, acumule pontos e participe da comunidade Club Nova Frota.

## MVP

A primeira versão do app terá:

- Login
- Tela inicial
- Fretes disponíveis
- Minhas ordens
- Anexar comprovante
- Perfil

## Tecnologia prevista

- Flutter
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Firebase Cloud Messaging
